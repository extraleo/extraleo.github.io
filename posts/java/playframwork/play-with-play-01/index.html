update:

It’s wrong if don’t set https_proxy. Create HTTPS_PROXY and HTTP_PROXY with windows way. If the proxy server doesn’t accept https, add http protocol like HTTPS_PROXY=http://proxyServer:port.

NO repository file, no vm options, just set HTTPS_PROXY rightly and your JAVA_OPTS.
