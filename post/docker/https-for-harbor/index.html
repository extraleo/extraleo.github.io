Exercise install Harbor and configure https.

prepare

  1. install docker by get-docker script.
  2. install compose by compose github install wiki
  3. install harbor by harbor wiki

lets getting CA

  1. mkdir /data/harbor-cert & cd /data/harbor-cert
  2. generate CA key sh openssl genrsa -out ca.key 4096

  3. CA crt

    openssl req -x509 -new -nodes -sha512 -days 3650 \
-subj "/CN=yourdomain.com" \
-key ca.key \
-out ca.crt

    lets getting server certificate

  4. server key

    openssl genrsa -out yourdomain.com.key 4096

  5. Generate a Certificate Signing request

    openssl req -sha512 -new \
-subj "/CN=yourdomain.com" \
-key yourdomain.com.key \
-out yourdomain.com.csr

  6. Generate Certificate for Harbor ```sh cat > v3.ext <<-EOF authorityKeyIdentifier=keyid,issuer basicConstraints=CA:FALSE keyUsage = digitalSignature, nonRepudiation, keyEncipherment, dataEncipherment extendedKeyUsage = serverAuth subjectAltName = @alt_names

[alt_names] DNS.1=yourdomain.com DNS.2=yourdomain DNS.3=hostname EOF

```shell
openssl x509 -req -sha512 -days 3650 \
    -extfile v3.ext \
    -CA ca.crt -CAkey ca.key -CAcreateserial \
    -in yourdomain.com.csr \
    -out yourdomain.com.crt

configure Harbor

  1. Convert server yourdomain.com.crt to yourdomain.com.cert: shell openssl x509 -inform PEM -in yourdomain.com.crt -out yourdomain.com.cert

  2. Delpoy yourdomain.com.cert, yourdomain.com.key, and ca.crt for Docker:

    # if /etc/docker/certs.d/yourdomain.com/ not exist, mkdir.
cp yourdomain.com.cert /etc/docker/certs.d/yourdomain.com/
cp yourdomain.com.key /etc/docker/certs.d/yourdomain.com/
cp ca.crt /etc/docker/certs.d/yourdomain.com/

The following illustrates a configuration with custom certificates:

/etc/docker/certs.d/
    └── yourdomain.com:port   
       ├── yourdomain.com.cert  <-- Server certificate signed by CA
       ├── yourdomain.com.key   <-- Server key signed by CA
       └── ca.crt               <-- Certificate authority that signed the registry certificate
  1. Edit harbor.yml, add certificate, harbor.yml should be like this:
#set hostnamehostname: yourdomain.com

http:
  port: 80
https:
  # https port for harbor, default is 443
  port: 443
  # The path of cert and key files for nginx
  certificate: /data/cert/yourdomain.com.crt
  private_key: /data/cert/yourdomain.com.key

  ......

Generate configuration files for Harbor:

docker-compose down -v # if harbor is running, down first, then run prepare script
./prepare
docker-compose up -d
  1. Valid https
  • Open browser, and open https://yourdomain.com, see the harbor login page.
  • docker login yourdomain.com should success.
